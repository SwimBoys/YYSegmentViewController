## Example

To run the example project, clone the repo, and run `pod install` from the Example directory first.

## 要求
- iOS 8.0+
- Xcode 9
- Swift

## 使用CocoaPods安装
```
pod 'LLSegmentViewController'
```
如果找不见，请输入源地址
```
source 'https://github.com/SwimBoys/YYSegmentViewControllerSpec.git'
```
